## 🎯 提示词优化技巧

### 1. 结构化描述
```
基础结构：
[主体] + [特征] + [视角/风格] + [技术参数] + [背景/环境]
```

### 2. 科学术语使用
- 使用准确的科学术语
- 明确量化参数（尺寸、比例、放大倍数等）
- 指定期刊风格（Nature、Science、Cell等）

### 3. 视觉效果控制
```
常用修饰词：
- 分辨率：高分辨率、超清、4K、8K
- 光照：均匀光照、对比度高、清晰阴影
- 视角：侧视图、俯视图、横截面、透视图
- 风格：科研期刊、教科书、学术海报、示意图
```

## ⚡ 高级提示词技巧

### 1. 多角度描述
```
"展示X的三个不同角度：正视图显示[特征A]，侧视图展示[特征B]，横截面揭示[特征C]，科学插图风格"
```

### 2. 比较与对比
```
"并列对比：左侧[状态A]，右侧[状态B]，相同尺度，实验前后对比，学术海报风格，箭头标注变化"
```

### 3. 动态过程
```
"[过程名称]动态序列，分步骤展示，时间轴标注，从[起始状态]到[终止状态]，流程图风格，每步清晰标注"
```
